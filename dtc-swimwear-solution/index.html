一、行业机会：季节性品类更需要「全年不停」的自动化

跨境 DTC 女装 / 泳装赛道销售高度季节性：旺季（约 3–8 月）流量与转化集中，淡季（约 10 月–次年 2 月）断崖下滑。 若只做「旺季广告爆推」，获客成本持续抬升，利润被挤压。跨境 DTC 泳装品牌在部署 欢迎系列、首购激励、售后复购、放弃浏览与放弃结账等全生命周期自动化后， 整体转化率提升约 60%、邮件送达率保持 99%+—— 全渠道生命周期运营可显著拉动增长。

EngageLab MA 在同一平台原生覆盖 Email / App Push / Web Push / SMS / WhatsApp， 从 0 建设时即按「场景选通道」编排——Email 承载内容与品牌叙事，Push / Web Push 抓黄金窗口， SMS / WhatsApp 覆盖高意向与高渗透市场——帮助品牌一次跑通可量化的全渠道增长闭环。

四、核心旅程详细编排（最优全渠道设计）

以下为面向跨境 DTC 女装 / 泳装的推荐编排：按场景选用最合适通道，一次覆盖 Email、Push、Web Push、SMS、WhatsApp。

旅程 1（P0）：弃单多通道挽回

业务痛点与通道逻辑

时尚品类弃单率约 78-85%。推荐编排：App Push 抓加购后黄金窗口，高意向按市场用 SMS（美/英/澳）或 WhatsApp（南欧/拉美），Email 承载尺码指南与评价等详述内容。

旅程组件配置

进入条件：事件: 加入购物车

提前退出：事件: 订单支付成功

转化统计：事件: 支付成功，监听 2 小时

再次进入：允许，间隔 1 天

条件分支：客单价 ≥ 1.5×AOV → 再按市场：SMS / WhatsApp；其余 → Email

预期效益（测试目标）

弃单恢复率目标区间 12-16%

高客单 + 即时通道路径通常更明显

旅程 2（P0）：浏览未加购召回

业务痛点与通道逻辑

独立站高意向浏览大量未加购；付费 Retargeting CTR 常低于 1%。Web Push 适合网页场景即时召回；已装 App 用户用 App Push 二次触达；Email 沉淀搭配与 UGC 内容。

旅程组件配置

进入条件：事件: 浏览商品详情页超过 60 秒

提前退出：事件: 加入购物车 或 支付成功

转化统计：事件: 加入购物车，监听 3 天

再次进入：允许，间隔 3 天

条件分支：是否已安装 App → App Push / Email

预期效益

浏览→加购转化相对提升 +30~50%（测试目标）

旅程 3（P0）：新客欢迎 + 首购激励

业务痛点与通道逻辑

欢迎与品牌故事最适合 Email（内容完整、可沉淀）；首购激励用 App Push 提高打开与行动；未转化再用 Email 尺码指南与热销榜收口。

旅程组件配置

进入条件：事件: 用户注册

提前退出：事件: 首次下单成功

转化统计：事件: 首次下单，监听 1 天

再次进入：不允许

预期效益

注册→首单相对 +20~35%（测试目标）

旅程 4（P1）：售后关怀 + 跨系列关联复购

业务痛点与通道逻辑

发货状态用 App Push 即时告知；保养 / 搭配指南用 Email 承载长内容；跨系列推荐再用 App Push 轻提醒 + Email 详述，推动泳装 ↔ 日常转化。

旅程组件配置

进入条件：事件: 订单发货

提前退出：事件: 再次下单

转化统计：事件: 再次下单，监听 14 天

条件分支：首单品类（泳装 / 日常）分流推荐

再次进入：允许，间隔 30 天

预期效益

首单 30 日复购相对 +25~40%（测试目标）

旅程 5（P1）：沉睡老客分层唤醒

业务痛点与通道逻辑

60/90/180 天分层：浅沉默用 App Push 轻触达，中深沉默用 Email 讲权益与礼包；末触达按市场选 SMS（美/英/澳）或 WhatsApp（南欧/拉美），避免一刀切折扣群发。

旅程组件配置

进入条件：属性: 最后下单距今超过 60 天

提前退出：事件: 下单成功

转化统计：事件: 下单成功，监听 7 天

再次进入：允许，间隔 60 天

条件分支：60 / 90 / 180 天分层；末触达再按市场：SMS / WhatsApp

预期效益

触达后 14 日内成交率目标 8-15%（按沉默深度递减）
